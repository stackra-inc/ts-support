# @abdokouta/ts-container-react

## 1.0.0

### Major Features

- 🎉 Initial release of @abdokouta/ts-container-react
- ⚛️ `ContainerProvider` component for React DI context
- 🎣 `useInject(token)` hook for resolving providers in components
- ❓ `useOptionalInject(token)` hook for optional provider resolution
- 🔧 `useContainer()` hook for raw `ContainerResolver` access
- 🔗 `ContainerContext` React context for advanced use cases
- 📐 `ContainerProviderProps` interface for type-safe provider props
- 🏗️ Full compatibility with `ApplicationContext` from
  `@abdokouta/ts-application`
